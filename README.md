# Proyecto-Modelado-y-Programacion
Coleccion de Videojuegos
Clases:
Biblioteca.java: Clase que maneja la biblioteca de juegos, incluyendo la lista maestra y las subcolecciones
DatosComunes.java: Clase que contiene listas comunes usadas en la aplicación
Estilos.java: Clase que define estilos de colores y fuentes para la interfaz gráfica
GameCard.java: Tarjeta visual para representar un juego en la interfaz gráfica 
Juego.java: Clase que representa un juego con sus atributos principales
MainFrame.java: Clase principal de la aplicación del gestor de videojuegos
Subcoleccion.java: Clase que representa una subcolección de juegos dentro de la biblioteca
